# Article Extraction Schema

기사 수집 시 추출할 필드와 길이 제한을 정의한다.
본문 전체는 수집하지 않는다.

---

## 필수 필드

| 필드 | 타입 | 최대 길이 | 설명 |
|------|------|-----------|------|
| `title` | string | 120자 | 기사 제목 |
| `url` | string | - | 원문 링크 |
| `source` | string | - | 언론사명 |
| `published` | string | - | 발행 시각 (ISO 8601) |

## 선택 필드

| 필드 | 타입 | 최대 길이 | 설명 |
|------|------|-----------|------|
| `summary` | string | 300자 | 기사 첫 2-3문장 |

---

## 규칙

- 본문(`body`) 전체는 수집하지 않는다.
- `summary`는 기사 첫 단락 또는 메타 description 태그에서 추출한다.
- `published`가 없는 기사는 수집 시각으로 대체한다.
- 120자를 초과하는 `title`은 120자에서 자른다.
- 300자를 초과하는 `summary`는 300자에서 자른다.

---

## 저장 예시

```json
{
  "title": "한화에어로스페이스, 폴란드에 K9 자주포 추가 수출 계약 체결",
  "url": "https://www.yonhapnews.co.kr/example",
  "source": "연합뉴스",
  "published": "2026-05-31T07:30:00+09:00",
  "summary": "한화에어로스페이스가 폴란드 국방부와 K9 자주포 48문 추가 공급 계약을 체결했다. 계약 규모는 약 6억 달러로 알려졌다."
}
```
