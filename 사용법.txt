================================
 방산 뉴스 대시보드 사용법
================================

[ 사전 조건 ]
- Python 3.10 이상 설치 필요
- 설치 확인: 시작 메뉴에서 "PowerShell" 열고 python --version 입력

[ 처음 시작할 때 (1회만) ]
1. [1_설치.bat] 더블클릭
2. 패키지 설치 완료 메시지 확인 후 창 닫기

[ 매일 사용할 때 ]
1. [2_실행.bat] 더블클릭
2. 브라우저에서 http://127.0.0.1:8000 자동으로 열림
3. "새로 수집" 버튼 클릭하면 최신 기사 수집
4. 사용 완료 후 실행 창 닫으면 종료

[ 화면 설명 ]
- 전체 / 국내 / 외신 버튼: 언론사 유형별 필터
- 언론사 이름 버튼: 특정 언론사만 보기
- 검색창: 제목·요약 키워드 검색
- 새로 수집: 지금 바로 최신 기사 가져오기
  (자동으로는 2시간마다 수집됨)

[ 수집 소스 ]
외신: Reuters, CNBC, Breaking Defense, The War Zone,
      Defense One, Defense News, New York Times(선택)
국내: 연합뉴스, 뉴시스, 조선비즈, 한국경제, 방산뉴스
